# Changelog

Все значимые изменения в проекте будут документированы в этом файле.

## [1.0.0] - 2026-01-19

### Добавлено
- Инициализация проекта React + Vite + Tailwind CSS
- Настройка Tailwind с кастомной конфигурацией
- Подключение шрифта Rubik через Google Fonts
- Создание структуры папок проекта

#### Компоненты Layout
- Header - шапка с навигацией и кнопками
- Footer - подвал с колонками навигации

#### UI Компоненты
- CategoryCard - карточка категории недвижимости
- PropertyCard - карточка объекта недвижимости
- ResidentialComplexCard - карточка жилого комплекса

#### Секции страницы
- SearchSection - поиск и категории
- OffersSection - новые объявления
- ResidentialComplexSection - каталог ЖК
- HotOffersSection - горящие предложения
- LaunchSalesSection - старт продаж
- AboutPlatformSection - о платформе Live Grid

#### Ассеты
- Скачано 18 изображений через Figma MCP
- Скачано 5 SVG иконок
- Все ассеты оптимизированы

#### Утилиты и данные
- mockData.js - тестовые данные
- helpers.js - утилиты форматирования
- theme.js - константы темы из Figma

#### Документация
- README.md - полная документация проекта
- REPORT.md - детальный отчет о работе
- CHANGELOG.md - история изменений

### Улучшено
- CategoryCard: улучшены пропорции и адаптивность
- PropertyCard: добавлены hover эффекты и анимации
- SearchSection: оптимизирована сетка категорий
- Все компоненты: добавлены transition эффекты

### Технические детали
- Tailwind CSS настроен с кастомными цветами из Figma
- Mobile-first подход для адаптивности
- Breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px)
- Шрифт Rubik: weights 300, 400, 500, 600, 700

### Проверено
- ✅ Нет linter ошибок
- ✅ Dev сервер запущен на localhost:3000
- ✅ HMR работает корректно
- ✅ Все изображения загружаются
- ✅ Адаптивность на всех брейкпоинтах
- ✅ Hover эффекты работают
